title: MotionLayoutProvider
---

Creates the React Portal where the elements are rendered during the transition and manages the internal state.

``` jsx
<MotionLayoutProvider />
```

### Props

| Name          |      type     |   required |
| ------------- | :-----------: | -----:   |
| children      |   Node        | true |
| debug     |   boolean    |   false |
